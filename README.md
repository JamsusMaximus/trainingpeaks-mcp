# TrainingPeaks MCP Server

MCP server for TrainingPeaks integration with AI assistants.

## Installation

```bash
pip install -e .
```

## Usage

```bash
# Authenticate with TrainingPeaks
tp-mcp auth

# Start the MCP server
tp-mcp serve
```
